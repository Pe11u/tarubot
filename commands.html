基本コマンド

  • t?tarubotボットの情報の表示
  • t?pingボットの応答速度（レイテンシ）を確認

管理コマンド

  • t?kick 項目１ 項目２ ユーザーをサーバーからキック
    t?kick @taru スパム
    • 項目１キックするユーザーのメンション
    • 項目２キックする理由
  • t?ban 項目１ 項目２ ユーザーをサーバーからBAN
    t?ban @taru 悪質なスパム
    • 項目１BANするユーザーのメンション
    • 項目２BANする理由
  • t?banlist過去にBANされたユーザーのIDを確認
  • t?unban 項目１ 過去にBANしたユーザーのBANを解除
    t?unban tarutaru_jpn
    • 項目１BAN解除するユーザーのユーザー名
  • t?timeout 項目１ 項目２ 項目３ ユーザーをタイムアウト
    t?timeout @taru 1d12h30m 荒らし
    • 項目１タイムアウトするユーザーのメンション
    • 項目２タイムアウトする長さ

      d=日, h=時間, m=分
      例: 1d12h30m（１日１２時間３０分）
    • 項目３（任意）タイムアウトする理由
  • t?untimeout 項目１ タイムアウト中のユーザーのタイムアウトを解除
    • 項目１タイムアウトを解除するユーザーのメンション
      t?untimeout @taru
  • t?clear 項目１ 項目２ 複数のメッセージを一括削除
    t?clear 100 @taru
    • 項目１削除するメッセージの数
    • 項目２（任意）削除するメッセージの送信者をメンションでフィルター
  • t?nickname 項目１ 項目２ 項目３ ユーザーのニックネームを変更
    t?nickname set @taru たる
    • 項目１オプション指定の項目です

      set: ニックネームを新しく設定
      reset: ニックネームをリセット
    • 項目２（条件）項目１でのオプションがsetの場合、新しく設定するユーザーのメンション
    • 項目２（条件）項目１でのオプションがsetの場合、新しく設定するニックネーム

情報コマンド

  • t?info 項目１ 項目２ ユーザー、チャンネル、サーバーの情報の表示
    t?info user @taru
    • 項目１オプション指定の項目です

      user: ユーザーの情報
      channel: チャンネルの情報
      server: サーバーの情報
    • 項目２（条件）項目１でのオプションがuserの場合、情報を閲覧するユーザーのメンション
  • t?icon 項目１ 項目２ ユーザー、サーバーのアイコンの表示
    t?icon user @taru
    • 項目１オプション指定の項目です

      user: ユーザーのアイコン
      server: サーバーのアイコン
    • 項目２（条件）項目１でのオプションがuserの場合、アイコンを取得するユーザーのメンション
  • t?role 項目１ 項目２ ユーザーの所属ロールおよびロールに所属しているユーザーを確認
    t?role check @taru
    • 項目１オプション指定の項目です

      member: ロールに所属しているユーザー
      check: ユーザーが所属するロール
    • 項目２（条件）項目１でのオプションがmemberの場合、確認したいユーザーのメンション
      checkの場合、確認したいロールのメンション（多数の人に通知が行きます）

技術コマンド

  • t?zishin 最新の日本での地震・津波情報の表示
    t?eq, t?earthquakeでも使用可能
  • t?train 最新の列車遅延状況の表示
  • t?qr項目１ 文字列をQRコードに変換
    t?qr たるボット
    • 項目１変換する文字列
  • t?readqr QRコード画像の読み取り
    画像を添付して実行
  • t?forex項目１ 項目２ 外貨と日本円の為替の表示
    t?forex usd 100
    • 項目１変換先の通貨コード
    • 項目２日本円での値段
  • t?ipconfig項目１ IPアドレス情報の表示
    t?ipconfig 8.8.8.8
    • 項目１IPv4, IPv6のIPアドレス
    • t?base64項目１ 項目２ 文字列をBase64に変換（Base32, Base58に置き換えても使用可能）
    t?base64 encode たるボット
    • 項目１オプション指定の項目です

      encode: 文字列をエンコード（符号化）
      decode: 文字列をデコード（復号）
    • 項目２変換する文字列
  • t?percent項目１ 項目２ 文字列をPercentに変換
    t?ppercent encode たるボット
    • 項目１オプション指定の項目です

      encode: 文字列をエンコード（符号化）
      decode: 文字列をデコード（復号）
    • 項目２変換する文字列
