基本コマンド

  • t?pingボットの応答速度（レイテンシ）を確認

管理コマンド

  • t?kick 項目１ 項目２ ユーザーをサーバーからキック
    t?kick @taru スパム
    • 項目１キックするユーザーのメンション
    • 項目２キックする理由
  • t?ban 項目１ 項目２ ユーザーをサーバーからBAN
    t?ban @taru 悪質なスパム
    • 項目１BANするユーザーのメンション
    • 項目２BANする理由
  • t?banlist過去にBANされたユーザーのIDを確認
  • t?unban 項目１ 過去にBANしたユーザーのBANを解除
    t?unban tarutaru_jpn
    • 項目１BAN解除するユーザーのユーザー名
  • t?timeout 項目１ 項目２ 項目３ ユーザーをタイムアウト
    t?timeout @taru 1d12h30m 荒らし
    • 項目１タイムアウトするユーザーのメンション
    • 項目２タイムアウトする長さ

      d=日, h=時間, m=分
      例: 1d12h30m（１日１２時間３０分）
    • 項目３（任意）タイムアウトする理由
  • t?untimeout 項目１ タイムアウト中のユーザーのタイムアウトを解除
    • 項目１タイムアウトを解除するユーザーのメンション
      t?untimeout @taru
  • t?clear 項目１ 項目２ 複数のメッセージを一括削除
    t?clear 100 @taru
    • 項目１削除するメッセージの数
    • 項目２（任意）削除するメッセージの送信者をメンションでフィルター
  • t?nickname 項目１ 項目２ 項目３ ユーザーのニックネームを変更
    t?nickname set @taru たる
    • 項目１オプション指定の項目です

      set: ニックネームを新しく設定
      reset: ニックネームをリセット
    • 項目２（条件）項目１でのオプションがsetの場合、新しく設定するユーザーのメンション
    • 項目２（条件）項目１でのオプションがsetの場合、新しく設定するニックネーム
  • t?say 項目１ ボットとしてメッセージを送信t?say たるボット
    • 項目１メッセージ内容
  • t?intent 項目１ 項目２ 項目３ ボットとして埋め込みメッセージを送信t?intent お知らせ たるボット green
    • 項目１タイトル
      空白が含まれる場合は''または""で囲う
    • 項目２デスクリプション（内容）
      空白が含まれる場合は''または""で囲う
    • 項目３（任意）メッセージの色
      HEXカラーコードまたは英語

情報コマンド

  • t?info 項目１ 項目２ ユーザー、チャンネル、サーバーの情報を表示
    t?info user @taru
    • 項目１オプション指定の項目です

      user: ユーザーの情報
      channel: チャンネルの情報
      server: サーバーの情報
    • 項目２（条件）項目１でのオプションがuserの場合、情報を閲覧するユーザーのメンション
  • t?icon 項目１ 項目２ ユーザー、サーバーのアイコンを表示
    t?icon user @taru
    • 項目１オプション指定の項目です

      user: ユーザーのアイコン
      server: サーバーのアイコン
    • 項目２（条件）項目１でのオプションがuserの場合、アイコンを取得するユーザーのメンション
  • t?role 項目１ 項目２ ユーザーの所属ロールおよびロールに所属しているユーザーを確認
    t?role check @taru
    • 項目１オプション指定の項目です

      member: ロールに所属しているユーザー
      check: ユーザーが所属するロール
    • 項目２（条件）項目１でのオプションがmemberの場合、確認したいユーザーのメンション
      checkの場合、確認したいロールのメンション（多数の人に通知が行きます）

技術コマンド

  • t?zishin 最新の日本での地震・津波情報を表示
    t?eq, t?earthquakeでも使用可能
  • t?train 最新の列車遅延状況を表示
  • t?qr 項目１ 文字列をQRコードに変換
    t?qr たるボット
    • 項目１変換する文字列
  • t?readqr QRコード画像の読み取り
    画像を添付して実行
  • t?forex 項目１ 項目２ 外貨と日本円の為替を表示
    t?forex usd 100
    • 項目１変換先の通貨コード
    • 項目２日本円での値段
  • t?ipconfig 項目１ IPアドレス情報を表示
    t?ipconfig 8.8.8.8
    • 項目１IPv4, IPv6のIPアドレス
    • t?base64 項目１ 項目２ 文字列をBase64に変換（Base32, Base58に置き換えても使用可能）
    t?base64 encode たるボット
    • 項目１オプション指定の項目です

      encode: 文字列をエンコード（符号化）
      decode: 文字列をデコード（復号）
    • 項目２変換する文字列
  • t?percent 項目１ 項目２ 文字列をPercentに変換
    t?ppercent encode たるボット
    • 項目１オプション指定の項目です

      encode: 文字列をエンコード（符号化）
      decode: 文字列をデコード（復号）
    • 項目２変換する文字列

VCコマンド

  • t?play 項目１ YouTube動画音声を再生
    t?play https://www.youtube.com/watch?v=dQw4w9WgXcQ
    • 項目１YouTube動画のリンク
  • t?skip 現在流れている曲をスキップ
  • t?leave ボイスチャットから退出
    強制退出させられるとボットが再起動するまで同じサーバーで使えなくなります

経済コマンド

  • t?work 働いて報酬を得る
  • t?crime 犯罪を犯して報酬を得るか、罰金を支払う
  • t?balance 現在の所持金を表示
  • t?leaderboard 項目１ サーバー内・ボット全体の所持金ランキングを表示
    t?leaderboard local
    • 項目１オプション指定の項目です

      local: サーバー内ランキング
      global: ボット全体ランキング
  • t?give 項目１ 項目２ 他ユーザーにお金を送金
    t?give @taru 5000
    • 項目１送金するユーザーのメンションまたはユーザー名
    • 項目２送金する金額
      allと入力して所持金全てを指定可能
  • t?blackjack 項目１ ボットとブラックジャックを遊ぶ
    t?blackjack 5000
    • 項目１賭ける金額
      allと入力して所持金全てを指定可能

その他

  • t?dice 項目１ 項目２ サイコロを回す
    t?dice 1d100 int
    • 項目１サイコロの面と回す回数の指定
      サイコロの面d回す回数
    • 項目２（任意）オプション指定の項目です

      int: 結果を全て整数値にする
      float: 結果をすべて小数点第5位を四捨五入した浮動小数点数（小数）にする
  • t?coinflip コインを投げる
  • t?janken ボットとじゃんけんを遊ぶ
  • t?today 今日が何の日か表示する

ジョークコマンド

  • t?fart VCに参加しておならする
  • t?kill 項目１ 項目２ ユーザーを殺す
    t?kill @taru thunder
    • 項目１殺すユーザーのメンション
    • 項目２（任意）オプション指定の項目です

      shoot, drowned, explode, fall, burn, lava, thunder, magic, froze, attack
