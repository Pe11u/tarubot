基本コマンド

  • t?help 利用可能なコマンドの一覧を表示
  • t?tarubot ボットの情報を表示
  • t?ping ボットの応答速度（レイテンシ）を確認

管理コマンド

  • t?kick 項目１ 項目２ ユーザーをサーバーからキック
    t?kick @taru スパム
    • 項目１キックするユーザーのメンション
    • 項目２キックする理由
  • t?ban 項目１ 項目２ ユーザーをサーバーからBAN
    t?ban @taru 悪質なスパム
    • 項目１BANするユーザーのメンション
    • 項目２BANする理由
  • t?banlist 過去にBANされたユーザーのIDを確認
  • t?unban 項目１ 過去にBANしたユーザーのBANを解除
    t?unban tarutaru_jpn
    • 項目１BAN解除するユーザーのユーザー名
  • t?timeout 項目１ 項目２ 項目３ ユーザーをタイムアウト
    t?timeout @taru 1d12h30m 荒らし
    • 項目１タイムアウトするユーザーのメンション
    • 項目２タイムアウトする長さ (d=日, h=時間, m=分)
    • 項目３（任意）タイムアウトする理由
  • t?untimeout 項目１ タイムアウト中のユーザーのタイムアウトを解除
    • 項目１タイムアウトを解除するユーザーのメンション
      t?untimeout @taru
  • t?clear 項目１ 項目２ 複数のメッセージを一括削除
    t?clear 100 @taru
    • 項目１削除するメッセージの数
    • 項目２（任意）削除するメッセージの送信者をメンションでフィルター
  • t?nickname 項目１ 項目２ 項目３ ユーザーのニックネームを変更
    t?nickname set @taru たる
    • 項目１オプション指定 (set: 新規, reset: リセット)
    • 項目２（条件）setの場合、変更するユーザーのメンション
    • 項目３（条件）setの場合、新しいニックネーム
  • t?say 項目１ ボットとしてメッセージを送信
    t?say たるボット
    • 項目１送信するメッセージ内容
  • t?intent 項目１ 項目２ 項目３ ボットとして埋め込みメッセージを送信
    t?intent お知らせ たるボット green
    • 項目１タイトル (空白は''または\"\")
    • 項目２内容 (空白は''または\"\")
    • 項目３（任意）メッセージの色 (HEXまたは英語名)

情報コマンド

  • t?info 項目１ 項目２ ユーザー、チャンネル、サーバーの情報を表示
    t?info user @taru
    • 項目１オプション (user, channel, server)
    • 項目２（条件）userの場合、対象ユーザーのメンション
  • t?icon 項目１ 項目２ ユーザー、サーバーのアイコンを表示
    t?icon user @taru
    • 項目１オプション (user, server)
    • 項目２（条件）userの場合、対象ユーザーのメンション
  • t?role 項目１ 項目２ ユーザーの所属ロールおよびロール内のユーザーを確認
    t?role check @taru
    • 項目１オプション (member, check)
    • 項目２（条件）memberの場合、対象ユーザー、checkの場合、対象ロール

技術コマンド

  • t?zishin 最新の日本での地震・津波情報を表示 (t?eq, t?earthquakeでも可)
  • t?train 最新の列車遅延状況を表示
  • t?qr 項目１ 文字列をQRコードに変換
    t?qr たるボット
    • 項目１変換する文字列
  • t?readqr QRコード画像の読み取り (画像を添付して実行)
  • t?forex 項目１ 項目２ 外貨と日本円の為替を表示
    t?forex usd 100
    • 項目１変換先の通貨コード
    • 項目２日本円の金額
  • t?ipconfig 項目１ IPアドレス情報を表示
    t?ipconfig 8.8.8.8
    • 項目１IPv4またはIPv6のIPアドレス
  • t?base64 項目１ 項目２ 文字列をBase64に変換 (Base32, Base58にも対応)
    t?base64 encode たるボット
    • 項目１オプション (encode: 符号化, decode: 復号)
    • 項目２変換する文字列
  • t?percent 項目１ 項目２ 文字列をPercentに変換
    t?ppercent encode たるボット
    • 項目１オプション (encode: 符号化, decode: 復号)
    • 項目２変換する文字列

VCコマンド

  • t?play 項目１ YouTube動画音声を再生
    t?play https://www.youtube.com/watch?v=dQw4w9WgXcQ
    • 項目１YouTube動画のリンク
  • t?skip 現在流れている曲をスキップ
  • t?leave ボイスチャットから退出 (強制退出時は再起動まで使用不可)

経済コマンド

  • t?work 働いて報酬を得る
  • t?crime 犯罪を犯して報酬または罰金
  • t?balance 現在の所持金を表示
  • t?leaderboard 項目１ サーバー内・全体の所持金ランキングを表示
    t?leaderboard local
    • 項目１オプション (local: サーバー内, global: 全体)
  • t?give 項目１ 項目２ 他ユーザーにお金を送金
    t?give @taru 5000
    • 項目１送金先のメンションまたは名前
    • 項目２送金金額 (allで全額指定可)
  • t?blackjack 項目１ ボットとブラックジャックを遊ぶ
    t?blackjack 5000
    • 項目１賭け金額 (allで全額指定可)

その他

  • t?dice 項目１ 項目２ サイコロを振る
    t?dice 1d100 int
    • 項目１サイコロの面数と回数 (例: 1d100)
    • 項目２（任意）オプション (int: 整数, float: 四捨五入浮動小数点)
  • t?coinflip コインを投げる
  • t?janken ボットとじゃんけんを遊ぶ
  • t?today 今日が何の日かを表示する

ジョークコマンド

  • t?fart VCに参加しておならをする
  • t?kill 項目１ 項目２ ユーザーをキルする
    t?kill @taru thunder
    • 項目１対象ユーザーのメンション
    • 項目２（任意）オプション (shoot, drowned, explode, fall, burn, lava, thunder, magic, froze, attack)
