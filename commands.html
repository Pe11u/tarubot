  • t?pingボットの応答速度（レイテンシ）を確認

管理コマンド

  • t?kick 項目１ 項目２ ユーザーをサーバーからキック
    t?kick @taru スパム
    • 項目１キックするユーザーのメンション
    • 項目２キックする理由
  • t?ban 項目１ 項目２ ユーザーをサーバーからBAN
    t?ban @taru 悪質なスパム
    • 項目１BANするユーザーのメンション
    • 項目２BANする理由
  • t?unban 項目１ 過去にBANしたユーザーのBANを解除
    t?unban tarutaru_jpn
    • 項目１BAN解除するユーザーのユーザー名
  • t?timeout 項目１ 項目２ 項目３ ユーザーをタイムアウト
    t?timeout @taru 1d12h30m 荒らし
    • 項目１タイムアウトするユーザーのメンション
    • 項目２タイムアウトする長さ

      d=日, h=時間, m=分
      例: 1d12h30m（１日１２時間３０分）
    • 項目３（任意）タイムアウトする理由
  • t?untimeout 項目１ タイムアウト中のユーザーのタイムアウトを解除
    • 項目１タイムアウトを解除するユーザーのメンション
      t?untimeout @taru
  • t?clear 項目１ 項目２ 複数のメッセージを一括削除
    t?kick 100 @taru
    • 項目１削除するメッセージの数
    • 項目２（任意）削除するメッセージの送信者をメンションでフィルター
  • t?nickname 項目１ 項目２ 項目３ ユーザーのニックネームを変更
    t?nickname set @taru たる
    • 項目１オプション指定の項目です

      set: ニックネームを新しく設定
      reset: ニックネームをリセット
    • 項目２（条件）項目１でのオプションがsetの場合、新しく設定するユーザーのメンション
    • 項目２（条件）項目１でのオプションがsetの場合、新しく設定するニックネーム

情報コマンド

  • t?info 項目１ 項目２ ユーザー、チャンネル、サーバーの情報を表示
    t?info user @taru
    • 項目１オプション指定の項目です

      user: ユーザーの情報
      channel: チャンネルの情報
      server: サーバーの情報
    • 項目２（条件）項目１でのオプションがuserの場合、情報を閲覧するユーザーのメンション
  • t?punycode 項目１ 項目２ 文字列をPunycodeに変換
    t?punycode encode たるボット
    • 項目１オプション指定の項目です

      encode: 文字列をエンコード（符号化）
      decode: 文字列をデコード（復号）
    • 項目２変換する文字列
